# ArcMenu Project
- https://extensions.gnome.org/extension/3628/arcmenu/

# Install

```
$ cp ./po/ja.mo ~/.local/share/gnome-shell/extensions/arcmenu@arcmenu.com/locale/ja/LC_MESSAGES/arcmenu.mo
```
